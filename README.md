# LunaLis

Sistema de gestión de laboratorio clínico con inteligencia artificial integrada.
Construido por un bacteriólogo que conoce el problema desde adentro.

## El problema

Los laboratorios clínicos en Colombia operan con procesos manuales o sistemas 
fragmentados que generan errores, retrasos y riesgos reales para los pacientes.
Un error en la identificación de una muestra o un resultado enviado tarde no es 
un problema administrativo — es un problema clínico.

## La solución

LunaLis es una plataforma web integral que centraliza toda la operación del 
laboratorio, desde la recepción del paciente hasta la entrega del resultado 
al médico tratante, con inteligencia artificial integrada en cada módulo.

## Módulos

- Recepción y registro de pacientes
- Facturación y cobro
- Trazabilidad de muestras en tiempo real
- Integración bidireccional con analizadores (ASTM / HL7)
- Validación y liberación de resultados
- Portal del médico tratante
- Portal del paciente
- Notificaciones por WhatsApp y correo
- Módulo veterinario
- Reportes y estadísticas

## IA integrada

- Asistente preanalítico: indica tubo, volumen, condiciones de ayuno y transporte
- Validación inteligente de resultados contra historial y valores de referencia
- Alertas predictivas de problemas operativos
- Asistente de consulta disponible en todo momento

## Stack técnico

- Frontend: React + TypeScript
- Backend: NestJS (Node.js)
- Base de datos: PostgreSQL
- Protocolos: ASTM E1394 / HL7 v2.x
- Notificaciones: WhatsApp Business API
